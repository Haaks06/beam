iPhone setup

Beam in your Share Sheet

iOS doesn't let a website register itself as a Share Sheet target the way Android does, so on iPhone Beam works through Apple Shortcuts. You build it once in about five minutes, then Beam shows up next to AirDrop whenever you share a link or a photo. No App Store, no account.

There are two shortcuts — one for links, one for photos. Each is a straight line of steps with no branching, so there's very little to get wrong. Build whichever you'll actually use; you don't need both.

Before you start

Open Beam on your PC and leave it showing its pairing code. That code is good for 10 minutes before anyone claims it, which is plenty of time to set this up and send something. The shortcut asks you for the code each time you use it, claims the pairing and sends the item back-to-back in one run — so you're never racing the session timer by hand.

“Beam Link to PC”

  1. Shortcuts app → + → name it Beam Link to PC.
  2. Tap the shortcut's settings (ⓘ) → Show in Share Sheet → on. Under “Share Sheet Types”, enable URLs and Safari web pages (add Text too if you want to share plain notes).
  3. Add action Ask for Input → Input Type: Text → Prompt: “Beam pairing code (from your PC's Beam window)”.
  4. Add action Set Variable → name it PairingCode.
  5. Add action Get Contents of URL:
    • URL: https://beamlot.com/pair/claim?pairingCode= then insert the PairingCode variable, then type &label=iPhone
    • Method: POST (no headers, no request body)
  6. Add action Get Dictionary Value → Key: token.
  7. Add action Set Variable → name it Token.
  8. Add action Get Contents of URL:
    • URL: https://beamlot.com/items/link?token= then insert the Token variable, then type &url= then insert the Shortcut Input variable
    • Method: POST
  9. Optionally add Show Notification → “Beamed!” at the end.
  10. Save. Then share a link from any app → Share → “Beam Link to PC” → type the code your PC is showing.

“Beam Photo to PC”

Same shape, with a different final step.

  1. Shortcuts app → + → name it Beam Photo to PC.
  2. Show in Share Sheet → on, enable Images only.
  3. Add action Ask for Input → Text → Prompt: “Beam pairing code (from your PC's Beam window)”.
  4. Set Variable → PairingCode.
  5. Get Contents of URL → same claim URL and method as step 5 above.
  6. Get Dictionary Value → Key: token.
  7. Set Variable → Token.
  8. Get Contents of URL:
    • URL: https://beamlot.com/items/photo?token= then insert the Token variable
    • Method: POST
    • Request Body: File, value = Shortcut Input
  9. Optionally add Show Notification → “Beamed!”.
  10. Save and use the same way — share a photo → “Beam Photo to PC” → type the code.

Good to know

  • A pairing code is single-use. If you mistype it or reuse one that's already claimed, the step fails visibly rather than silently sending nowhere.
  • Nothing sensitive is stored inside the shortcut — the code you type each time is single-use and dead within 10 minutes even if unused. So it's safe to share the shortcut itself with someone without scrubbing anything first.
  • If your PC is mid-session with something else, click Start over in Beam first to get a fresh code.
